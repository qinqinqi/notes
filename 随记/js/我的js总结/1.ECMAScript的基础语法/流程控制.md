# 一、流程控制

## １．什么是流程

代码执行的顺序和方式

## ２．流程控制

用一些特殊的语句控制程序按照我们想要的方式选择执行或者重复执行

## 3.顺序结构

*代码正常的执行顺序	按照引入的顺序一块一块语句进行执行，在每一块中从上往下一条一条执行

## 4.选择结构

### *分支结构

> * 单路分支
>
>   if(表达式){当表达式为真的时候要执行的代码}
>
>   	<script>
>   		var grade=prompt("请输入成绩：")
>   		if(grade>=60){
>   			console.log("及格")
>   		}
>   		console.log("不及格")
>   	</script>

> * 双路分支
>
>   if(表达式){当表达式为真的时候要执行的代码}
>
>   else{当表达式为假的时候要执行的代码}
>
>   	<script>
>   		var grade=prompt("请输入成绩：")
>   		if(grade>=60){
>   			console.log("及格")
>   		}
>   		else{
>   			console.log("不及格")
>   		}
>   	</script>

> * 多路分支
>
>   if(表达式1){当表达式1为真时要执行的代码}
>
>   else if(表达式2){当表达式2为真时要执行的代码}
>
>   .
>
>   .
>
>   .
>
>   	<script>

>   	</script>

### *条件结构

```javascript
switch(表达式){
  case 值1：当表达式的值为值1的时候要执行的代码;
    break;
  case 值2：当表达式的值为值2的时候要执行的代码;
    break;
    ...
  default：当表达式的值和所有值都不匹配的时候要执行的代码;
}
```
